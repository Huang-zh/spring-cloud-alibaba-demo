## spring cloud alibaba 实践项目

> Springboot版本：2.2.3.RELEASE<br/>
  Spring Cloud Alibaba版本：2.2.0.RELEASE

#### 包含如下已实现的功能：
* 服务治理
* 集成sentinel对接口进行熔断、降级、限流
* gateway网关统一代理微服务，并实现统一鉴权和限流
* zipkin+sleuth实现链路追踪
* 集成RocketMQ实现多种消息发送机制，并通过事务消息实现分布式事务的最终一致性
